Project Overview 

In this project, I worked on the Superstore dataset to understand how a real business performs using data. The main idea was to explore the data step by step and discover useful insights about sales, profit, customers, and regions.

I started by loading the dataset and getting familiar with its structure. Then I cleaned the data and prepared it for analysis by fixing date formats and creating new useful features like year, month, and profit ratio. After that, I analyzed the data using grouping, pivot tables, and visualizations to find patterns and trends.
